TECHNOLOGIES UTILISÉES
================================

Backend:
  - Node.js + Express.js
  - PostgreSQL
  - JWT (jsonwebtoken)
  - Bcrypt.js
  - CORS
  - dotenv

Frontend:
  - React 18
  - React Router DOM v6
  - Axios
  - CSS vanilla (pas de framework)

Base de données:
  - PostgreSQL 12+
  - 4 tables (utilisateurs, vehicules, clients, locations)
